## Description
This project is my first AI, made with Scratch. It learned to play Hangman using the French dictionary. As a training method, it was rewarded when it found a correct letter in the word and penalized when it made a mistake or lost. The more frequently the letter appeared in the word, the greater the reward. It had knowledge of the wordâ€™s length, the letters already tried, and the letters yet to be tested.
